8. expo-document-picker  // npx expo install expo-document-picker  for image pic from device. -
9. expo-location  // npx expo install expo-location  for location. -
10. expo-permissions  // npx expo install expo-permissions  for location permission.

12. expo-secure-store  // npx expo install expo-secure-store  for secure store. -
